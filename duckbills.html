Duckbills






Duckbills the app that helps room mates know their responsibilities so they cant duck on the bills.
Using the Database the room mate that is the Admin creates and account for the house.
The Admin creates and account where they can add or delete room mates. The Admin is the only person that can add or remove room mates. This also collects email addresses so that reminders can be sent when a bill is due.
Any room mate can add a house bill and who is responsible for it. They can also put how much of that bill they are responsible for and when it is due. Future expansion would allow bill photo to be added as well as auto email when bill is close.
Roomates can see all the current bills and the one close to due date here. this application could be used for landlord applications as well.
